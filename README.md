# EnglishConversationPractice

> 英会話練習アプリ

<p>My goal is to create an app that enhances my English conversation abilities.</p>
<p>英会話の練習をしたい、自分のためのアプリです。</p>

## 基本機能

<ul>
<li>質問・回答例の登録</li>
<li>質問・回答例の編集</li>
<li>質問・回答例の削除</li>
<li>練習（ランダム）</li>
<li>練習（もう一度）</li>
<li>練習（リスト）</li>
<li>設定（音声）</li>
<li>設定（カテゴリ名）</li>
<li>設定（リスト新規登録）</li>
<li>設定（リスト編集）</li>
<li>設定（リスト削除）</li>
</ul>
